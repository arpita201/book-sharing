Book Sharing System

A web-based Book Sharing System developed using PHP, MySQL, HTML, CSS, and JavaScript.

Features
- User Authentication
- Dashboard
- Add Book
- Upload Book Image
- Search Books
- Borrow Books
- My Books
- My Requests
- Book Reviews
- Profile
- Edit/Delete Books

Technologies
- HTML5
- CSS3
- JavaScript
- PHP
- MySQL
- XAMPP

How to Run
1. Clone repository
2. Import bookshare_db.sql
3. Start Apache & MySQL
4. Open localhost/book-sharing
